How to Use

FlowWrite provides AI-powered writing suggestions as you type:

  • Tab key: Accept the entire suggestion
  • Ctrl + Right Arrow: Accept only the next word of the suggestion (press and hold Ctrl, then press Right Arrow). Press repeatedly to accept multiple words one by one.
  • Click: Click directly on a suggestion to accept it
  • Esc key: Dismiss the current suggestion

The click-to-accept feature is especially useful in web applications where the Tab key is already assigned to other actions.

Try the new "Dual" presentation mode to see suggestions both inline and in a popup simultaneously!
